# X Post Best Practices

## Hook Patterns That Work

### Curiosity Gap
Create information asymmetry - promise value, deliver after engagement.
```
I've been building AI apps for 3 years. Here's the one thing I wish I knew on day 1:
```

### Contrarian Take
Challenge conventional wisdom (authentically).
```
Hot take: API keys are a security nightmare that we all just accepted.
```

### Specific Numbers
Concrete beats vague every time.
```
I went from 500 to 50,000 followers in 6 months. Not with threads. Not with engagement pods.
```

### Problem Statement
Lead with pain your audience feels.
```
Managing 5 different API keys for AI models is insane.
```

### Bold Claim
Make a statement worth engaging with.
```
Claude Code can now generate images, make API calls, and access real-time data.
```

## What NOT to Open With

| Avoid | Why | Instead |
|-------|-----|---------|
| "I'm excited to announce..." | Boring, self-focused | Lead with the value |
| "Thread:" as first word | No hook, easy to scroll past | Hook first, "Thread:" at end |
| Vague statements | Nothing to engage with | Specific claims/numbers |
| Questions without context | Feels like engagement bait | Set up, then ask |
| ALL CAPS | Quality penalty | Emphasis sparingly |

## Post Structure

### Single Tweet
```
[Hook - first line grabs attention]

[Value - teaches, entertains, or provokes]

[CTA - drives engagement]
```

### Thread Format
```
Tweet 1: Bold claim/hook + "Here's how:" or "Thread:"
Tweet 2-6: Supporting points with examples
Tweet 7: Summary + CTA + "Reply with your thoughts"
```

**After posting:** Reply to your own thread with bonus value.

## Engagement Triggers

End posts with these to maximize replies (+13.5 weight):

- "What would you build with this?"
- "Reply if you want the code"
- "What's your biggest pain point with X?"
- "Hot take or facts?"
- "What am I missing?"
- "Tag someone who needs this"

Implicit questions work better than explicit asks.

## Formatting Tips

- Line breaks improve readability
- Lists work well for scanning
- Keep under 280 chars for full visibility (no "See more")
- If longer, make first 280 chars hook-worthy

## Media Guidelines

| Type | Boost | Tips |
|------|-------|------|
| Image | High | High contrast, minimal text |
| Video | High | Hook in first 3 sec, 50%+ completion |
| None | Lower | Only if content is strong enough |

Optimal image size: 1200x675 (X preview ratio)

## Hashtag Rules

- 0-2 relevant hashtags maximum
- Never more than 3 (dampening penalty)
- Often none is better
- Use only if genuinely relevant

## Timing Strategy

- Post when YOUR audience is active (check analytics)
- Engage immediately after posting
- Reply to EVERY comment in first 30 minutes
- Don't post and disappear

## Red Flags to Avoid

- Asking for likes/retweets explicitly (engagement bait)
- 4+ hashtags
- ALL CAPS throughout
- High following/follower ratio (> 0.6)
- Automation patterns (same time daily, identical formats)
- Triggering "Show less" or reports

## Thread Engagement Strategy

1. Tweet 1 must stand alone and hook
2. Each tweet valuable independently
3. Final tweet: clear CTA + invite replies
4. Reply to your own thread with bonus value
5. Engage with EVERY comment within 30 minutes
